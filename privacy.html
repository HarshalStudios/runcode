Privacy Policy

Last Updated: July, 2026

1. Introduction

Welcome to RunCoder. RunCoder is an Android code editor designed for students, developers, and programming enthusiasts. This Privacy Policy explains what information may be collected when you use the application, how it is used, and how we protect your information. By using RunCoder, you agree to the practices described in this Privacy Policy.

2. Your Code and Projects

Your programming projects, source code, folders, and files remain your property. RunCoder stores your projects locally on your Android device using Android's Storage Access Framework (SAF). The application only accesses files and folders that you explicitly choose. RunCoder does not scan your device or access files outside the locations you authorize.

3. Cloud Code Execution

When you choose to run or execute code, RunCoder securely sends the following information to a remote code execution service (Judge0 Community Edition):

  • Programming language
  • Source code
  • Standard input (if provided)

This information is transmitted solely to compile and execute your program and return the execution result. RunCoder does not associate submitted source code with your identity or user account. Your local project files remain stored on your device.

4. Information We Collect

RunCoder itself does not require user registration and does not collect personal information such as:

  • Name
  • Email address
  • Phone number
  • Postal address
  • Government identification
  • Contacts
  • Messages
  • Photos
  • Videos

However, anonymous technical information may be collected through integrated Google services to improve application quality. This may include:

  • Crash reports
  • Diagnostics
  • Device model
  • Android version
  • Application version
  • Anonymous usage events
  • Advertising identifiers (used by Google AdMob)

5. Third-Party Services

RunCoder uses the following trusted third-party services:

  • Firebase Analytics: Used to understand anonymous application usage and improve future updates.
  • Firebase Crashlytics: Used to receive anonymous crash reports and diagnose application issues.
  • Google AdMob: Displays advertisements in the free version of RunCoder. Google may collect advertising identifiers and related information according to Google's own Privacy Policy.
  • Google Play Billing: Processes purchases securely through Google Play. RunCoder never receives or stores your payment card information.
  • Judge0 Community Edition: Used to compile and execute code when you press the Run button. Only the source code, selected programming language, and optional standard input are transmitted for execution.

6. Permissions

RunCoder requests only the permissions necessary for its intended functionality:

  • Internet: Required for Cloud code execution, Firebase Analytics, Firebase Crashlytics, Google AdMob, and Google Play Billing.
  • Storage Access Framework (SAF): Used only after you explicitly choose files or folders to open or save. RunCoder does not request broad storage permissions.

7. Advertising

The free version of RunCoder may display advertisements provided by Google AdMob. Advertising partners may collect limited device identifiers as described in Google's Privacy Policy. RunCoder does not sell your personal information.

8. Data Security

We take reasonable measures to protect your information. Local project files remain stored on your device. Network communication uses encrypted HTTPS connections. RunCoder follows Android security best practices. Sensitive payment information is processed by Google Play, not by RunCoder.

9. Data Retention

RunCoder does not maintain user accounts or cloud storage for projects. Projects remain stored on your device unless you delete them. Crash reports and analytics data are retained according to the policies of Firebase and Google services.

10. Children's Privacy

RunCoder is intended for students, educators, and developers. The application does not knowingly collect personal information from children.

11. Changes to This Privacy Policy

This Privacy Policy may be updated periodically. Any changes will be published through an updated version of this document.
